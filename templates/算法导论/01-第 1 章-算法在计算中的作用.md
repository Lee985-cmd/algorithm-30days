# 📚 算法导论第 1 章 - 算法在计算中的作用

## 1.1 什么是算法？

### 正式定义

```
算法（Algorithm）是良定义的（well-defined）计算过程：
- 输入：取某个值或值的集合作为输入
- 输出：产生某个值或值的集合作为输出
- 过程：一系列计算步骤，将输入转换为输出
```

### 算法的五个特性

1. **有穷性（Finiteness）**
   - 算法必须在执行有限步之后终止
   - 每一步必须在有限时间内完成

2. **确定性（Definiteness）**
   - 算法的每一步骤必须确切地定义
   - 无歧义，相同的输入总是得到相同的输出

3. **输入（Input）**
   - 有 0 个或多个输入
   - 这些输入取自特定的对象集合

4. **输出（Output）**
   - 有 1 个或多个输出
   - 输出与输入有某种特定关系

5. **可行性（Effectiveness）**
   - 所有操作都是足够基本的
   - 原则上可以用纸笔在有限时间内完成

## 1.4 算法示例：排序问题

### 问题描述

```
输入：一个数列 (a₁, a₂, ..., aₙ)
输出：数列的一个排列 (a'₁, a'₂, ..., a'ₙ)，使得 a'₁ ≤ a'₂ ≤ ... ≤ a'ₙ

例如：
输入：(31, 41, 59, 26, 41, 58)
输出：(26, 31, 41, 41, 58, 59)
```

### 插入排序算法

```javascript
/**
 * 插入排序
 * @param {number[]} arr - 待排序数组
 */
function insertionSort(arr) {
    for (let j = 2; j <= arr.length; j++) {
        const key = arr[j];
        let i = j - 1;
        
        // 将比 key 大的元素向后移动
        while (i > 0 && arr[i] > key) {
            arr[i + 1] = arr[i];
            i = i - 1;
        }
        
        arr[i + 1] = key;
    }
    
    return arr;
}

// 测试
const arr = [5, 2, 4, 6, 1, 3];
console.log('排序前:', arr);
console.log('排序后:', insertionSort(arr));
```

### 算法正确性证明

```
循环不变式（Loop Invariant）：

在第 j 次迭代开始时，子数组 A[1..j-1] 
包含原来在位置 1 到 j-1 的元素，且已排好序。

证明三步骤：

1. 初始化：
   j = 2 时，A[1..1] 只有一个元素，显然有序 ✓

2. 保持：
   如果某次迭代前 A[1..j-1] 有序
   那么插入 A[j] 后，A[1..j] 仍然有序 ✓

3. 终止：
   当 j = n+1 时，循环结束
   A[1..n] 有序，即整个数组有序 ✓

证毕！
```

## 1.6 分治法简介

### 基本思想

```
分治法（Divide and Conquer）：

1. 分解（Divide）：
   将原问题分解为若干个规模较小的子问题

2. 解决（Conquer）：
   递归地求解这些子问题
   如果子问题足够小，直接求解

3. 合并（Combine）：
   将子问题的解合并成原问题的解
```

### 归并排序示例

```javascript
/**
 * 归并排序 - 分治法的经典例子
 */
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }
    
    // 分解
    const mid = Math.floor(arr.length / 2);
    const left = arr.slice(0, mid);
    const right = arr.slice(mid);
    
    // 递归解决
    const sortedLeft = mergeSort(left);
    const sortedRight = mergeSort(right);
    
    // 合并
    return merge(sortedLeft, sortedRight);
}

/**
 * 合并两个有序数组
 */
function merge(left, right) {
    const result = [];
    let i = 0, j = 0;
    
    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            result.push(left[i]);
            i++;
        } else {
            result.push(right[j]);
            j++;
        }
    }
    
    // 添加剩余元素
    return result.concat(left.slice(i)).concat(right.slice(j));
}

// 测试
const arr = [38, 27, 43, 3, 9, 82, 10];
console.log('排序后:', mergeSort(arr));
```

### 时间复杂度分析

```
归并排序的时间复杂度：T(n) = 2T(n/2) + O(n)

根据主定理：T(n) = O(n log n)

对比插入排序的 O(n²)：
- n=1000 时，n log n ≈ 10000，n² = 1000000
- 归并排序快 100 倍！
```
