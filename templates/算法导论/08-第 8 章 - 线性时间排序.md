# 📚 算法导论第 8 章 - 线性时间排序

## 8.1 排序的下界

### 比较排序的极限

```
定理：
任何比较排序算法在最坏情况下都需要 Ω(n log n) 次比较

证明思路：
- n 个元素有 n! 种可能的排列
- 每次比较有 2 种结果
- k 次比较最多区分 2^k 种情况
- 需要 2^k ≥ n!
- 推出 k ≥ log(n!) = Ω(n log n)
```

### 突破 O(n log n)

```
要在线性时间内排序，必须：
不使用比较操作！

三种线性时间排序：
1. 计数排序（Counting Sort）
2. 基数排序（Radix Sort）
3. 桶排序（Bucket Sort）
```

## 8.2 计数排序

### 适用场景

```
输入：n 个整数，范围在 [0, k]
条件：k = O(n)
时间复杂度：O(n + k) = O(n)
```

### 代码实现

```javascript
/**
 * 计数排序
 */
function countingSort(arr, k) {
    const n = arr.length;
    
    // 计数数组
    const count = new Array(k + 1).fill(0);
    
    // 统计每个元素出现的次数
    for (let i = 0; i < n; i++) {
        count[arr[i]]++;
    }
    
    // 重建原数组
    let index = 0;
    for (let i = 0; i <= k; i++) {
        while (count[i] > 0) {
            arr[index++] = i;
            count[i]--;
        }
    }
    
    return arr;
}

// 测试
const arr = [4, 2, 2, 8, 3, 3, 1];
console.log('排序前:', arr);
console.log('排序后:', countingSort(arr, 10));
// 输出：[1, 2, 2, 3, 3, 4, 8] ✓

// 时间复杂度：O(n + k)
// 空间复杂度：O(k)
```

## 8.3 基数排序

### 思想

```
从低位到高位，依次对每一位进行排序

使用稳定排序（如计数排序）作为子程序

时间复杂度：O(d(n + k))
其中 d 是位数，k 是每位的取值范围
```

### 代码实现

```javascript
/**
 * 基数排序
 */
function radixSort(arr, d = 10) {
    if (arr.length === 0) return arr;
    
    const max = Math.max(...arr);
    
    // 从个位开始，对每一位进行计数排序
    for (let exp = 1; Math.floor(max / exp) > 0; exp *= 10) {
        countingSortByDigit(arr, exp);
    }
    
    return arr;
}

/**
 * 按指定位数进行计数排序
 */
function countingSortByDigit(arr, exp) {
    const n = arr.length;
    const output = new Array(n);
    const count = new Array(10).fill(0);
    
    // 统计当前位的数字出现次数
    for (let i = 0; i < n; i++) {
        const digit = Math.floor(arr[i] / exp) % 10;
        count[digit]++;
    }
    
    // 计算累积计数
    for (let i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }
    
    // 从后向前遍历，保证稳定性
    for (let i = n - 1; i >= 0; i--) {
        const digit = Math.floor(arr[i] / exp) % 10;
        output[count[digit] - 1] = arr[i];
        count[digit]--;
    }
    
    // 复制回原数组
    for (let i = 0; i < n; i++) {
        arr[i] = output[i];
    }
}

// 测试
const arr = [170, 45, 75, 90, 802, 24, 2, 66];
console.log('排序前:', arr);
console.log('排序后:', radixSort(arr));
// 输出：[2, 24, 45, 66, 75, 90, 170, 802] ✓
```

## 8.4 桶排序

### 思想

```
将区间 [0, 1) 分成 n 个大小相同的桶
将 n 个数放入对应的桶中
对每个桶内排序（通常用插入排序）
依次输出所有桶

平均时间复杂度：O(n)
```

### 代码实现

```javascript
/**
 * 桶排序
 */
function bucketSort(arr) {
    if (arr.length === 0) return arr;
    
    const n = arr.length;
    const buckets = Array.from({ length: n }, () => []);
    
    // 分配到各个桶
    for (let i = 0; i < n; i++) {
        const index = Math.floor(arr[i] * n);
        buckets[index].push(arr[i]);
    }
    
    // 对每个桶排序
    for (let i = 0; i < n; i++) {
        buckets[i].sort((a, b) => a - b);
    }
    
    // 合并所有桶
    const result = [];
    for (let i = 0; i < n; i++) {
        result.push(...buckets[i]);
    }
    
    return result;
}

// 测试
const arr = [0.897, 0.565, 0.656, 0.1234, 0.665, 0.3434];
console.log('排序前:', arr);
console.log('排序后:', bucketSort(arr));
// 输出：[0.1234, 0.3434, 0.565, 0.656, 0.665, 0.897] ✓
```
