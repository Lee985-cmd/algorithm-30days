## 5.1 雇佣问题

```
问题描述：
你要雇佣一个新的办公助理
每天面试一个候选人
如果当前候选人比现有的好，就雇佣他/她

每次雇佣的成本很高
问：期望的雇佣次数是多少？
```

## 5.2 指示器随机变量

```javascript
// 用指示器随机变量分析期望

// Xᵢ = I{第 i 个候选人被雇佣}
// E[Xᵢ] = P(第 i 个候选人被雇佣) = 1/i

// 总雇佣次数的期望：
// E[X] = Σ(i=1 to n) E[Xᵢ] = Σ(i=1 to n) 1/i = ln n + O(1)

// 所以期望雇佣次数约为 ln n
```

## 5.3 随机算法

```javascript
// 随机排列数组（Fisher-Yates 洗牌算法）
function randomizeArray(arr) {
    const n = arr.length;
    
    for (let i = 0; i < n; i++) {
        // 在 [i, n-1] 中随机选一个位置
        const j = i + Math.floor(Math.random() * (n - i));
        
        // 交换 arr[i] 和 arr[j]
        [arr[i], arr[j]] = [arr[j], arr[i]];
    }
    
    return arr;
}

// 每个排列出现的概率都是 1/n!
```
