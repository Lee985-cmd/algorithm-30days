# 📚 算法导论第 30 章 - 线性规划

## 30.1 线性规划基础

```
线性规划问题：
在满足一组线性约束的条件下，最大化或最小化线性目标函数。

标准形式：
最大化：c₁x₁ + c₂x₂ + ... + cₙxₙ
约束于：
    a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
    a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
    ...
    aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ ≤ bₘ
    x₁, x₂, ..., xₙ ≥ 0
```

## 30.2 单纯形算法

```javascript
/**
 * 单纯形算法实现（简化版）
 * 用于求解标准形式的线性规划问题
 */
class Simplex {
    constructor(A, b, c) {
        this.m = A.length;      // 约束数
        this.n = A[0].length;   // 变量数
        this.A = A;             // 约束矩阵
        this.b = b;             // 右端项
        this.c = c;             // 目标函数系数
    }
    
    solve() {
        // 初始化基变量和非基变量
        const basicVars = new Array(this.m);
        const nonBasicVars = [];
        
        for (let i = 0; i < this.n; i++) {
            nonBasicVars.push(i);
        }
        
        // 迭代优化
        while (true) {
            // 找进基变量（选择正系数的非基变量）
            let enterVar = -1;
            for (let j = 0; j < nonBasicVars.length; j++) {
                if (this.c[nonBasicVars[j]] > 0) {
                    enterVar = nonBasicVars[j];
                    break;
                }
            }
            
            if (enterVar === -1) break;  // 已最优
            
            // 找出基变量
            let leaveVar = -1;
            let minRatio = Infinity;
            
            for (let i = 0; i < this.m; i++) {
                if (this.A[i][enterVar] > 0) {
                    const ratio = this.b[i] / this.A[i][enterVar];
                    if (ratio < minRatio) {
                        minRatio = ratio;
                        leaveVar = i;
                    }
                }
            }
            
            if (leaveVar === -1) {
                throw new Error('问题无界');
            }
            
            // 旋转操作
            this.pivot(leaveVar, enterVar);
        }
        
        // 计算最优值
        let optimalValue = 0;
        for (let i = 0; i < this.n; i++) {
            if (this.c[i] > 0) {
                optimalValue += this.c[i] * this.b[i];
            }
        }
        
        return optimalValue;
    }
    
    pivot(row, col) {
        // 归一化主元行
        const pivotElement = this.A[row][col];
        this.b[row] /= pivotElement;
        
        for (let j = 0; j < this.n; j++) {
            this.A[row][j] /= pivotElement;
        }
        
        // 消去其他行的该列
        for (let i = 0; i < this.m; i++) {
            if (i !== row && this.A[i][col] !== 0) {
                const factor = this.A[i][col];
                this.b[i] -= factor * this.b[row];
                
                for (let j = 0; j < this.n; j++) {
                    this.A[i][j] -= factor * this.A[row][j];
                }
            }
        }
        
        // 更新目标函数
        const factor = this.c[col];
        for (let j = 0; j < this.n; j++) {
            this.c[j] -= factor * this.A[row][j];
        }
    }
}

// 测试示例
// 最大化：3x₁ + x₂ + 2x₃
// 约束于：
//     x₁ + x₂ + 3x₃ ≤ 30
//     2x₁ + 2x₂ + 5x₃ ≤ 24
//     4x₁ + x₂ + 2x₃ ≤ 36
//     x₁, x₂, x₃ ≥ 0

const A = [
    [1, 1, 3],
    [2, 2, 5],
    [4, 1, 2]
];

const b = [30, 24, 36];
const c = [3, 1, 2];

const simplex = new Simplex(A, b, c);
const result = simplex.solve();

console.log('最优值:', result);
```

## 30.3 对偶理论

```
原问题（P）：
最大化 cᵀx
约束于 Ax ≤ b, x ≥ 0

对偶问题（D）：
最小化 bᵀy
约束于 Aᵀy ≥ c, y ≥ 0

弱对偶定理：
对于任意可行解 x 和 y，有 cᵀx ≤ bᵀy

强对偶定理：
如果原问题有最优解，那么对偶问题也有最优解，
且最优值相等。
```
