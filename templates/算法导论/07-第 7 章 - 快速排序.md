# 📚 算法导论第 7 章 - 快速排序

## 7.1 快速排序算法

### 分治思想

```
快速排序三步骤：

1. 分解（Divide）：
   选择一个枢轴（pivot）
   将数组分为两部分：
   - 左边：都 ≤ pivot
   - 右边：都 ≥ pivot

2. 解决（Conquer）：
   递归地对左右两部分排序

3. 合并（Combine）：
   不需要额外操作，已经有序
```

### 代码实现

```javascript
/**
 * 快速排序
 */
function quickSort(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        // 分区
        const pivotIndex = partition(arr, low, high);
        
        // 递归排序左右两部分
        quickSort(arr, low, pivotIndex - 1);
        quickSort(arr, pivotIndex + 1, high);
    }
    
    return arr;
}

/**
 * 分区操作（Lomuto 方案）
 */
function partition(arr, low, high) {
    const pivot = arr[high];  // 选择最后一个元素作为枢轴
    let i = low - 1;          // i 指向小于 pivot 区域的最后一个元素
    
    for (let j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            [arr[i], arr[j]] = [arr[j], arr[i]];
        }
    }
    
    // 将 pivot 放到中间
    [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
    
    return i + 1;  // 返回 pivot 的位置
}

// 测试
const arr = [10, 80, 30, 90, 40, 50, 70];
console.log('排序前:', arr);
console.log('排序后:', quickSort(arr));
// 输出：[10, 30, 40, 50, 70, 80, 90] ✓
```

## 7.2 性能分析

### 最坏情况

```
当数组已经有序时：
每次分区都极不平衡
T(n) = T(n-1) + O(n) = O(n²)

例如：[1, 2, 3, 4, 5, 6, 7]
每次都选到最小或最大元素作为 pivot
```

### 最好情况

```
当每次都能平分时：
T(n) = 2T(n/2) + O(n) = O(n log n)

例如：每次都正好选到中位数
```

### 平均情况

```
随机情况下：
期望时间复杂度：O(n log n)

实际表现：
快速排序是实践中最快的排序算法之一！
```

## 7.3 随机化快速排序

```javascript
/**
 * 随机化快速排序
 */
function randomizedQuickSort(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        // 随机选择 pivot
        const randomPivotIndex = Math.floor(Math.random() * (high - low + 1)) + low;
        [arr[randomPivotIndex], arr[high]] = [arr[high], arr[randomPivotIndex]];
        
        const pivotIndex = partition(arr, low, high);
        randomizedQuickSort(arr, low, pivotIndex - 1);
        randomizedQuickSort(arr, pivotIndex + 1, high);
    }
    
    return arr;
}

// 期望时间复杂度：O(n log n)
// 避免了最坏情况
```
