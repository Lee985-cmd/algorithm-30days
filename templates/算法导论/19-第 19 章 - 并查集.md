# 📚 算法导论第 19 章 - 并查集

## 19.1 并查集操作

```
并查集（Disjoint Set Union, DSU）支持三种操作：
- MAKE-SET(x): 创建一个只包含 x 的集合
- FIND-SET(x): 返回 x 所在集合的代表元素
- UNION(x, y): 合并 x 和 y 所在的集合

应用：Kruskal 算法、连通分量检测
```

## 19.2 实现（路径压缩 + 按秩合并）

```javascript
/**
 * 并查集
 */
class UnionFind {
    constructor(n) {
        this.parent = new Array(n);
        this.rank = new Array(n).fill(0);
        
        // 初始化：每个元素自成一个集合
        for (let i = 0; i < n; i++) {
            this.parent[i] = i;
        }
    }
    
    // 查找（带路径压缩）
    findSet(x) {
        if (this.parent[x] !== x) {
            this.parent[x] = this.findSet(this.parent[x]);  // 路径压缩
        }
        return this.parent[x];
    }
    
    // 合并（按秩合并）
    union(x, y) {
        const rootX = this.findSet(x);
        const rootY = this.findSet(y);
        
        if (rootX === rootY) return false;  // 已经在同一集合
        
        // 按秩合并：将秩小的树挂到秩大的树下
        if (this.rank[rootX] < this.rank[rootY]) {
            this.parent[rootX] = rootY;
        } else if (this.rank[rootX] > this.rank[rootY]) {
            this.parent[rootY] = rootX;
        } else {
            this.parent[rootY] = rootX;
            this.rank[rootX]++;
        }
        
        return true;
    }
    
    // 判断是否在同一集合
    connected(x, y) {
        return this.findSet(x) === this.findSet(y);
    }
}

// 测试
const uf = new UnionFind(10);
uf.union(0, 1);
uf.union(2, 3);
uf.union(4, 5);
uf.union(0, 2);

console.log('0 和 3 是否连通:', uf.connected(0, 3)); // true ✓
console.log('0 和 4 是否连通:', uf.connected(0, 4)); // false ✓
```

## 19.3 时间复杂度分析

```
使用两种优化后：
- m 次操作的总时间复杂度：O(m α(n))
- 其中 α(n) 是反阿克曼函数，增长极慢
- 对于所有实际应用，α(n) ≤ 4

因此可以认为是近似 O(1) 的操作！
```
