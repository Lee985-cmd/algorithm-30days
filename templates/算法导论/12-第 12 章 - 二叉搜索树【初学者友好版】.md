## 🔑 核心概念

### 什么是二叉搜索树？

```
两个规则（必须同时满足）：

规则 1：左小右大
- 左子树的所有节点 < 根节点
- 右子树的所有节点 > 根节点

规则 2：递归定义
- 左右子树也是二叉搜索树

例子：
       8
      / \
     3   10
    / \    \
   1   6    14
   
验证：
✓ 节点 3 的左边 1 < 3，右边 6 > 3
✓ 节点 8 的左边都 < 8，右边都 > 8
✓ 整棵树都是 BST
```

## 🛠️ 基本操作

### 节点定义

```javascript
/**
 * 二叉搜索树节点
 */
class TreeNode {
    constructor(val) {
        this.val = val;
        this.left = null;
        this.right = null;
    }
}
```

### 查找操作（Search）

```javascript
/**
 * 查找值为 key 的节点
 * 
 * @param {TreeNode} root - 根节点
 * @param {number} key - 目标值
 * @returns {TreeNode|null}
 */
function searchBST(root, key) {
    // 基本情况：空树或找到了
    if (root === null || root.val === key) {
        return root;
    }
    
    // 在左子树或右子树中找
    if (key < root.val) {
        return searchBST(root.left, key);
    } else {
        return searchBST(root.right, key);
    }
}

// 迭代版本（更省空间）
function searchBSTIterative(root, key) {
    let current = root;
    
    while (current !== null) {
        if (key === current.val) {
            return current;
        } else if (key < current.val) {
            current = current.left;
        } else {
            current = current.right;
        }
    }
    
    return null;
}

// ⏱️ 时间复杂度：O(h)，h 是树高
// 平均 O(log n)，最坏 O(n)
```

### 插入操作（Insert）

```javascript
/**
 * 插入新值
 * 
 * @param {TreeNode} root - 根节点
 * @param {number} val - 要插入的值
 * @returns {TreeNode} 新的根节点
 */
function insertIntoBST(root, val) {
    // 基本情况：空位置，创建新节点
    if (root === null) {
        return new TreeNode(val);
    }
    
    // 递归插入到左子树或右子树
    if (val < root.val) {
        root.left = insertIntoBST(root.left, val);
    } else if (val > root.val) {
        root.right = insertIntoBST(root.right, val);
    }
    // 如果相等，根据需求决定是否插入
    
    return root;
}

// ⏱️ 时间复杂度：O(h)
```

### 删除操作（Delete）⭐最难

```javascript
/**
 * 删除指定值的节点
 * 
 * @param {TreeNode} root - 根节点
 * @param {number} key - 要删除的值
 * @returns {TreeNode} 新的根节点
 */
function deleteFromBST(root, key) {
    if (root === null) return null;
    
    // 1. 找到要删除的节点
    if (key < root.val) {
        root.left = deleteFromBST(root.left, key);
    } else if (key > root.val) {
        root.right = deleteFromBST(root.right, key);
    } else {
        // 2. 找到了，开始删除
        
        // 情况 1: 叶子节点（没有孩子）
        if (root.left === null && root.right === null) {
            return null;
        }
        
        // 情况 2: 只有一个孩子
        if (root.left === null) {
            return root.right;
        }
        if (root.right === null) {
            return root.left;
        }
        
        // 情况 3: 有两个孩子（最复杂！）
        // 策略：找右子树的最小节点（后继）来替换
        const minNode = findMin(root.right);
        
        // 用后继的值替换当前节点
        root.val = minNode.val;
        
        // 删除后继节点
        root.right = deleteFromBST(root.right, minNode.val);
    }
    
    return root;
}

/**
 * 找最小值（最左边的节点）
 */
function findMin(root) {
    while (root.left !== null) {
        root = root.left;
    }
    return root;
}

// ⏱️ 时间复杂度：O(h)
```

### 删除操作的三种情况图解

```
情况 1: 删除叶子节点
     5              5
    / \            /
   3   7   →     3
  /
 2

直接去掉，简单！

情况 2: 删除只有一个孩子的节点
     5              5
    / \            /
   3   7   →     2
  /
 2

让它的孩子接替它的位置

情况 3: 删除有两个孩子的节点（最复杂！）
     8                    10
    / \                  /  \
   3   10   →          3    14
  / \   \              / \   /
 1   6   14           1  6 12
        /
       12

步骤：
1. 找右子树的最小值（10 的后继是 12）
2. 用 12 替换 10
3. 删除原来的 12
```

## ⏱️ 复杂度分析

### 最好情况（完美平衡）

```
       8
      / \
     4   12
    / \  / \
   2  6 10 14
   
树高 h = log₂n
查找：O(log n) ⚡
插入：O(log n) ⚡
删除：O(log n) ⚡
```

### 最坏情况（退化成链表）

```
1
 \
  2
   \
    3
     \
      4
       \
        5

树高 h = n
查找：O(n) ❌
插入：O(n) ❌
删除：O(n) ❌

完全退化成链表！
```

### 平均情况

```
随机插入的情况下：
期望树高 h = O(log n)

所以平均是 O(log n) ✨
```

## 🎯 应用实例

### 验证二叉搜索树

```javascript
/**
 * 验证一棵树是否是 BST
 * 
 * @param {TreeNode} root
 * @returns {boolean}
 */
function isValidBST(root) {
    function helper(node, lower, upper) {
        if (node === null) return true;
        
        const val = node.val;
        
        // 当前节点必须在 (lower, upper) 范围内
        if (val <= lower || val >= upper) return false;
        
        // 递归检查左右子树
        if (!helper(node.right, val, upper)) return false;
        if (!helper(node.left, lower, val)) return false;
        
        return true;
    }
    
    return helper(root, -Infinity, Infinity);
}

// LeetCode 98. 验证二叉搜索树
```

### 找第 k 小的元素

```javascript
/**
 * BST 中第 k 小的元素
 * 
 * @param {TreeNode} root
 * @param {number} k
 * @returns {number}
 */
function kthSmallest(root, k) {
    let count = 0;
    let result = null;
    
    function inorder(node) {
        if (node === null || result !== null) return;
        
        // 中序遍历：左 → 根 → 右
        inorder(node.left);
        
        count++;
        if (count === k) {
            result = node.val;
            return;
        }
        
        inorder(node.right);
    }
    
    inorder(root);
    return result;
}

// ⏱️ 时间复杂度：O(h + k)
```

### 常见问题

**Q: BST 和散列表哪个更好？**

```
✅ 标准答案：

BST 优点：
1. 支持有序遍历（中序遍历）
2. 支持范围查询（找 [min, max] 之间的所有值）
3. 支持找第 k 小、前驱、后继等操作
4. 不需要处理哈希冲突

散列表优点：
1. 平均 O(1)，更快
2. 实现简单（如果用库函数）

选择建议：
- 需要有序性 → BST
- 只需要快速查找 → Hash Table
```
