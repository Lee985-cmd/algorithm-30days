# 📚 算法导论第 26 章 - NP 完全性

## 26.1 P vs NP 问题

```
P 类问题：
可以在多项式时间内解决的问题

NP 类问题：
可以在多项式时间内验证解的问题

NP 完全问题：
最难的 NP 问题，如果有一个能在多项式时间内解决，
那么所有 NP 问题都能在多项式时间内解决

P = NP ? （千禧年大奖难题之一）
```

## 26.3 近似算法

```javascript
/**
 * 顶点覆盖的近似算法
 * 近似比：2
 */
function approximateVertexCover(graph) {
    const C = new Set();
    const coveredEdges = new Set();
    
    while (coveredEdges.size < graph.edges.length) {
        // 选一条未覆盖的边
        let edge = null;
        for (const e of graph.edges) {
            if (!coveredEdges.has(e)) {
                edge = e;
                break;
            }
        }
        
        if (edge === null) break;
        
        // 将两个端点都加入覆盖集
        C.add(edge.u);
        C.add(edge.v);
        
        // 标记所有与这两个点相邻的边为已覆盖
        for (const e of graph.edges) {
            if (e.u === edge.u || e.v === edge.u ||
                e.u === edge.v || e.v === edge.v) {
                coveredEdges.add(e);
            }
        }
    }
    
    return C;
}

// 这是一个 2-近似算法
// 最优解的大小至少是匹配数，我们选了 2 倍匹配数
```
