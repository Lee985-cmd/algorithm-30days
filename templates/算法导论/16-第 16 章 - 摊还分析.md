## 16.1 聚合分析

```
示例：栈操作

操作：
- PUSH(S, x): O(1)
- POP(S): O(1)
- MULTIPOP(S, k): O(k)

n 个操作的总代价：O(n)
每个操作的摊还代价：O(1)
```

## 16.2 核算法

```
思想：
给不同操作分配不同的"摊还代价"

栈操作示例：
- PUSH: 收费 2 元（1 元用于入栈，1 元存起来用于将来弹出）
- POP: 收费 0 元（用之前存的钱支付）
- MULTIPOP: 收费 0 元

总摊还代价：O(n)
```

## 16.3 势能法

```javascript
/**
 * 动态表的势能分析
 * 
 * 势能函数 Φ(Dᵢ) = 2 × (元素个数 - 表大小/2)
 * 
 * 插入操作的摊还代价：
 * - 不触发扩展：ĉᵢ = cᵢ + ΔΦ = 1 + 2 = 3
 * - 触发扩展：ĉᵢ = cᵢ + ΔΦ = i + (2 - i) = 3
 * 
 * 所以每次插入的摊还代价是 O(1)
 */

class DynamicArray {
    constructor() {
        this.array = new Array(1);
        this.size = 0;
        this.capacity = 1;
    }
    
    push(val) {
        if (this.size === this.capacity) {
            // 扩展为原来的 2 倍
            this.resize(this.capacity * 2);
        }
        
        this.array[this.size++] = val;
    }
    
    resize(newCapacity) {
        const newArray = new Array(newCapacity);
        
        for (let i = 0; i < this.size; i++) {
            newArray[i] = this.array[i];
        }
        
        this.array = newArray;
        this.capacity = newCapacity;
    }
}

// n 次 push 操作的总时间复杂度：O(n)
// 每次 push 的摊还时间复杂度：O(1)
```
