### 10.1 栈（Stack）- 后进先出的智慧

#### 生活中的例子

**场景 1：弹夹**

```
子弹装进弹夹：
第一颗子弹最先装进去
但最后才能打出来

最后一颗子弹最后装进去
但最先打出来

这就是"后进先出"（LIFO）！
```

**场景 2：浏览器后退按钮**

```
你浏览网页的顺序：
1. 打开百度
2. 点击新闻 → 新浪
3. 点击体育 → 网易
4. 点击科技 → 36Kr

现在点"后退"：
回到网易 ← 最后访问的页面
再点"后退"：
回到新浪
再点"后退"：
回到百度

浏览器用栈记录你的浏览历史！
```

**说人话定义：**

```
栈 = 后进先出（LIFO）的数据结构

操作：
✓ 压栈（push）：把元素放到栈顶
✓ 出栈（pop）：从栈顶取出元素
✓ 查看栈顶（peek）：看看栈顶是什么，但不取出

就像叠盘子：
新盘子永远放在最上面
取盘子也只能从最上面取
```

#### 代码实现

```javascript
/**
 * 栈的实现
 */
class Stack {
    constructor() {
        this.items = [];
    }
    
    /**
     * 压栈 - 把元素放到栈顶
     */
    push(element) {
        this.items.push(element);
    }
    
    /**
     * 出栈 - 从栈顶取出元素
     */
    pop() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items.pop();
    }
    
    /**
     * 查看栈顶
     */
    peek() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items[this.items.length - 1];
    }
    
    /**
     * 判断是否为空
     */
    isEmpty() {
        return this.items.length === 0;
    }
    
    /**
     * 获取栈的大小
     */
    size() {
        return this.items.length;
    }
    
    /**
     * 清空栈
     */
    clear() {
        this.items = [];
    }
}

// 测试
const stack = new Stack();
stack.push(1);
stack.push(2);
stack.push(3);

console.log(stack.peek());    // 3（栈顶）
console.log(stack.pop());     // 3
console.log(stack.pop());     // 2
console.log(stack.size());    // 1
console.log(stack.isEmpty()); // false
```

#### 详细执行过程

```
初始：stack = []

═══════════════════════════════════
push(1)
═══════════════════════════════════
items = [1]
       ↑
     栈顶

═══════════════════════════════════
push(2)
═══════════════════════════════════
items = [1, 2]
          ↑
        栈顶

═══════════════════════════════════
push(3)
═══════════════════════════════════
items = [1, 2, 3]
             ↑
           栈顶

═══════════════════════════════════
peek()
═══════════════════════════════════
返回 items[2] = 3
items 不变：[1, 2, 3]

═══════════════════════════════════
pop()
═══════════════════════════════════
返回 3
items = [1, 2]
          ↑
        栈顶

═══════════════════════════════════
pop()
═══════════════════════════════════
返回 2
items = [1]
       ↑
     栈顶
```

#### 时间复杂度

```
push: O(1) - 在数组末尾添加
pop: O(1) - 从数组末尾删除
peek: O(1) - 访问最后一个元素
isEmpty: O(1)
size: O(1)

所有操作都是 O(1)！非常快！
```

#### 实际应用

```
1. 函数调用栈
   → 程序调用函数的过程
   → 递归的本质

2. 表达式求值
   → 计算器
   → 编译器解析表达式

3. 括号匹配
   → 检查代码中的括号是否配对

4. 撤销操作
   → Ctrl+Z
   → 每次撤销都是弹出最近的操作

5. DFS（深度优先搜索）
   → 用栈实现非递归的 DFS
```

### 10.2 队列（Queue）- 先进先出的公平

#### 生活中的例子

**场景 1：排队买票**

```
电影院买票：

第一个来的人排第一
第一个买到票

后来的人排后面
按顺序买票

这就是"先进先出"（FIFO）！
```

**场景 2：打印任务**

```
办公室打印机：

你先发送打印任务
同事后发送

打印机先打印你的
再打印同事的

先来后到，公平！
```

**说人话定义：**

```
队列 = 先进先出（FIFO）的数据结构

操作：
✓ 入队（enqueue）：在队尾添加元素
✓ 出队（dequeue）：从队头取出元素
✓ 查看队头（front）：看看队头是什么

就像排队：
新来的排后面
前面的先服务
```

#### 代码实现

```javascript
/**
 * 队列的实现
 */
class Queue {
    constructor() {
        this.items = [];
    }
    
    /**
     * 入队 - 在队尾添加
     */
    enqueue(element) {
        this.items.push(element);
    }
    
    /**
     * 出队 - 从队头取出
     */
    dequeue() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items.shift();  // 删除第一个元素
    }
    
    /**
     * 查看队头
     */
    front() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items[0];
    }
    
    /**
     * 判断是否为空
     */
    isEmpty() {
        return this.items.length === 0;
    }
    
    /**
     * 获取队列大小
     */
    size() {
        return this.items.length;
    }
    
    /**
     * 清空队列
     */
    clear() {
        this.items = [];
    }
}

// 测试
const queue = new Queue();
queue.enqueue(1);
queue.enqueue(2);
queue.enqueue(3);

console.log(queue.front());   // 1（队头）
console.log(queue.dequeue()); // 1
console.log(queue.dequeue()); // 2
console.log(queue.size());    // 1
```

```

═══════════════════════════════════

       ↑
     队头/队尾

═══════════════════════════════════

═══════════════════════════════════
items = [1, 2]
       ↑     ↑
     队头   队尾

═══════════════════════════════════
enqueue(3)
═══════════════════════════════════
items = [1, 2, 3]
       ↑        ↑
     队头      队尾

═══════════════════════════════════

═══════════════════════════════════
返回 items[0] = 1
items 不变

═══════════════════════════════════

═══════════════════════════════════
返回 1
items = [2, 3]
       ↑     ↑
     队头   队尾

═══════════════════════════════════

═══════════════════════════════════
返回 2
items = [3]
       ↑
     队头/队尾
```

#### 时间复杂度

```
enqueue: O(1) - 在数组末尾添加
dequeue: O(n) - 删除第一个元素，后面的要前移 ❌
front: O(1)
isEmpty: O(1)

问题：dequeue 太慢了！

优化：用两个指针或循环队列
可以让 dequeue 也变成 O(1)！
```

#### 循环队列（优化版）

```javascript
/**
 * 循环队列 - 优化的队列实现
 */
class CircularQueue {
    constructor(capacity) {
        this.capacity = capacity;  // 容量
        this.items = new Array(capacity);
        this.front = 0;   // 队头指针
        this.rear = 0;    // 队尾指针
        this.count = 0;   // 元素个数
    }
    
    enqueue(element) {
        if (this.isFull()) {
            return false;
        }
        
        this.items[this.rear] = element;
        this.rear = (this.rear + 1) % this.capacity;  // 循环
        this.count++;
        return true;
    }
    
    dequeue() {
        if (this.isEmpty()) {
            return null;
        }
        
        const value = this.items[this.front];
        this.front = (this.front + 1) % this.capacity;  // 循环
        this.count--;
        return value;
    }
    
    front() {
        if (this.isEmpty()) {
            return null;
        }
        return this.items[this.front];
    }
    
    isEmpty() {
        return this.count === 0;
    }
    
    isFull() {
        return this.count === this.capacity;
    }
    
    size() {
        return this.count;
    }
}

// 测试
const cq = new CircularQueue(5);
cq.enqueue(1);
cq.enqueue(2);
cq.enqueue(3);

console.log(cq.dequeue());  // 1
console.log(cq.dequeue());  // 2
cq.enqueue(4);  // 可以复用前面的空间！
```

#### 实际应用

```
1. BFS（广度优先搜索）
   → 用队列实现层序遍历

2. 任务调度
   → 操作系统进程调度
   → 打印队列

3. 消息队列
   → 生产者 - 消费者模式
   → 异步处理

4. 缓冲区
   → 键盘输入缓冲
   → 网络数据包缓冲

5. 限流
   → 限制请求速率
```

### 10.3 链表（Linked List）- 灵活的线性结构

#### 生活中的例子

**场景 1：寻宝游戏**

```
每个线索指向下一个线索的位置：

线索 1 → "去操场"
         ↓
线索 2 → "去图书馆"
         ↓
线索 3 → "去食堂"
         ↓
终点！

这就是链表的思想！
每个节点包含：
- 宝藏（数据）
- 下一个线索的地址（指针）
```

**场景 2：火车车厢**

```
一列火车：
车头 → 车厢 1 → 车厢 2 → 车厢 3 → ...

每节车厢通过挂钩连接
可以随时增加或减少车厢

这也是链表！
```

**说人话定义：**

```
链表 = 由节点组成的线性数据结构

每个节点包含：
1. 数据域：存储实际数据
2. 指针域：指向下一个节点

特点：
✓ 动态大小（可随时增删）
✓ 插入删除快（O(1)）
✗ 随机访问慢（O(n)）

对比数组：
数组：连续内存，随机访问快，插入删除慢
链表：分散内存，随机访问慢，插入删除快
```

#### 单向链表实现

```javascript
/**
 * 链表节点
 */
class ListNode {
    constructor(val) {
        this.val = val;
        this.next = null;  // 指向下一个节点
    }
}

/**
 * 单向链表
 */
class LinkedList {
    constructor() {
        this.head = null;  // 头节点
        this.size = 0;     // 链表长度
    }
    
    /**
     * 在末尾添加元素
     */
    append(val) {
        const newNode = new ListNode(val);
        
        if (this.head === null) {
            // 空链表，新节点就是头
            this.head = newNode;
        } else {
            // 找到最后一个节点
            let current = this.head;
            while (current.next !== null) {
                current = current.next;
            }
            current.next = newNode;
        }
        
        this.size++;
    }
    
    /**
     * 在指定位置插入
     */
    insertAt(index, val) {
        if (index < 0 || index > this.size) {
            return false;
        }
        
        const newNode = new ListNode(val);
        
        if (index === 0) {
            // 在头部插入
            newNode.next = this.head;
            this.head = newNode;
        } else {
            // 找到前一个节点
            let prev = null;
            let current = this.head;
            for (let i = 0; i < index; i++) {
                prev = current;
                current = current.next;
            }
            
            // 插入新节点
            prev.next = newNode;
            newNode.next = current;
        }
        
        this.size++;
        return true;
    }
    
    /**
     * 删除指定值的节点
     */
    delete(val) {
        if (this.head === null) {
            return false;
        }
        
        if (this.head.val === val) {
            // 删除头节点
            this.head = this.head.next;
            this.size--;
            return true;
        }
        
        // 找前一个节点
        let prev = this.head;
        let current = this.head.next;
        
        while (current !== null) {
            if (current.val === val) {
                prev.next = current.next;
                this.size--;
                return true;
            }
            prev = current;
            current = current.next;
        }
        
        return false;  // 没找到
    }
    
    /**
     * 查找元素
     */
    find(val) {
        let current = this.head;
        let index = 0;
        
        while (current !== null) {
            if (current.val === val) {
                return index;
            }
            current = current.next;
            index++;
        }
        
        return -1;  // 没找到
    }
    
    /**
     * 获取指定位置的元素
     */
    get(index) {
        if (index < 0 || index >= this.size) {
            return null;
        }
        
        let current = this.head;
        for (let i = 0; i < index; i++) {
            current = current.next;
        }
        
        return current.val;
    }
    
    /**
     * 打印链表
     */
    print() {
        const values = [];
        let current = this.head;
        
        while (current !== null) {
            values.push(current.val);
            current = current.next;
        }
        
        console.log(values.join(' -> '));
    }
}

// 测试
const list = new LinkedList();
list.append(1);
list.append(2);
list.append(3);
list.print();  // 1 -> 2 -> 3

list.insertAt(1, 1.5);
list.print();  // 1 -> 1.5 -> 2 -> 3

list.delete(2);
list.print();  // 1 -> 1.5 -> 3

console.log(list.find(1.5));  // 1
console.log(list.get(2));     // 3
```

#### 详细执行过程

```
初始：list.head = null

═══════════════════════════════════
append(1)
═══════════════════════════════════
创建节点：[1|next:null]
head → [1|next:null]

═══════════════════════════════════
append(2)
═══════════════════════════════════
创建节点：[2|next:null]
找到最后一个节点（值为 1）
1.next = [2|next:null]

head → [1|next:─┬─] → [2|next:null]

═══════════════════════════════════
append(3)
═══════════════════════════════════
创建节点：[3|next:null]
找到最后一个节点（值为 2）
2.next = [3|next:null]

head → [1] → [2] → [3] → null

═══════════════════════════════════
insertAt(1, 1.5)
═══════════════════════════════════
在索引 1 插入 1.5

创建节点：[1.5|next:null]
找到索引 0 的节点（值为 1）
1.5.next = 1.next（即 2）
1.next = 1.5

head → [1] → [1.5] → [2] → [3] → null

═══════════════════════════════════
delete(2)
═══════════════════════════════════
找到值为 2 的节点的前一个（1.5）
1.5.next = 2.next（即 3）

head → [1] → [1.5] → [3] → null
```

#### 时间复杂度

```
访问：O(n) - 要从头遍历
查找：O(n) - 要遍历
插入：
  - 头部：O(1) ✓
  - 尾部：O(n) - 要遍历到尾部
  - 中间：O(n) - 要找位置
删除：
  - 头部：O(1) ✓
  - 其他：O(n) - 要找前一个节点

对比数组：
        访问  查找  插入  删除
数组：  O(1) O(n) O(n)  O(n)
链表：  O(n) O(n) O(1)* O(1)*
*指头部操作
```

### 10.4 三种数据结构对比

| 特性 | 栈 | 队列 | 链表 |
|------|-----|------|------|
| 原则 | LIFO | FIFO | 顺序链接 |

| 应用 | 函数调用、撤销 | BFS、任务调度 | 动态数据 |
