# 📚 算法导论 Part III - 红黑树超详解

## 第 13 章：自动平衡的魔法树 - 红黑树

### 13.1 为什么需要红黑树？

#### 二叉搜索树的问题

```
回忆一下二叉搜索树（BST）：

最好的情况（平衡）：
        4
      /   \
     2     6
    / \   / \
   1   3 5   7

高度：3
查找时间：O(log n) ✓

最坏的情况（退化）：
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
           \
            7

高度：7
查找时间：O(n) ❌ 退化成链表了！

问题：
普通的 BST 可能会退化
导致性能从 O(log n) 变成 O(n)

怎么办？
→ 需要一种能自动保持平衡的树！
→ 这就是红黑树！
```

#### 红黑树的诞生

```
1972 年，两位计算机科学家：
- Rudolf Bayer（鲁道夫·拜尔）
- Leonidas J. Guibas（莱奥尼达斯·吉巴斯）

发明了这种数据结构

因为每个节点要么是红色，要么是黑色
所以叫"红黑树"

它的神奇之处：
✓ 自动保持平衡
✓ 查找、插入、删除都是 O(log n)
✓ 实际应用广泛（C++ STL、Java TreeMap、Linux 内核）
```

### 13.2 红黑树的五条规则（交通规则）

#### 核心性质

```
红黑树是一棵二叉搜索树，但多了 5 条规则：

规则 1：每个节点是红色或黑色
→ 就像交通灯，只有两种颜色

规则 2：根节点是黑色
→ 老大必须是黑色的

规则 3：叶子节点（NIL 节点）是黑色
→ 所有空的位置都是黑色的叶子

规则 4：红色节点的子节点必须是黑色
→ 不能有两个连续的红色节点
→ 就像红灯后面必须跟绿灯

规则 5：从任一节点到其所有叶子节点的路径上，
       包含相同数量的黑色节点
→ 这叫"黑高"（black-height）
→ 保证没有哪条路径特别长
```

#### 图示说明

```
一棵合法的红黑树：

         4B
       /    \
     2R      6R
    /  \    /  \
   1B  3B  5B  7B
  / \  / \ / \ / \
 BB BB BB BB BB BB BB BB

标注：
- 数字：节点的值
- R：红色（Red）
- B：黑色（Black）
- BB：黑色的 NIL 叶子（通常省略不画）

验证规则：
✓ 规则 1：每个节点非红即黑
✓ 规则 2：根节点 4 是黑色
✓ 规则 3：所有 NIL 叶子是黑色
✓ 规则 4：没有连续的红色（2R 的孩子都是黑色，6R 也是）
✓ 规则 5：每条路径都有 2 个黑色节点
  4B→2R→1B→BB: 黑节点数 = 2
  4B→2R→3B→BB: 黑节点数 = 2
  4B→6R→5B→BB: 黑节点数 = 2
  4B→6R→7B→BB: 黑节点数 = 2

完美符合所有规则！✓
```

#### 为什么这 5 条规则能保证平衡？

```
关键在规则 5（黑高相同）+ 规则 4（红不相邻）

推论：
从根到叶子的最长路径 ≤ 最短路径的 2 倍

证明：
最短路径：全是黑色节点
最长路径：红黑交替（因为不能有连续红色）

所以：
最长路径 ≤ 2 × 最短路径

这意味着：
树的高度最多是最小高度的 2 倍
所以树是"大致平衡"的！

虽然不是完全平衡
但足够保证 O(log n) 的性能！
```

### 13.3 基本操作：旋转

#### 为什么要旋转？

```
当插入或删除节点时
可能会违反红黑树的规则

需要通过旋转来重新平衡

旋转就像"换座位"
不改变二叉搜索树的性质
但改变了树的结构
```

#### 左旋（Left Rotate）

```
旋转前：
    x
     \
      y
     / \
    a   b

以 x 为支点左旋：

    y
   /
  x
   \
    a

详细过程：
1. y 成为新的根
2. x 变成 y 的左孩子
3. y 原来的左孩子 a 变成 x 的右孩子

代码实现：
```

```javascript
/**
 * 左旋
 * @param {TreeNode} x - 旋转的支点节点
 */
function leftRotate(x) {
    const y = x.right;  // y 是 x 的右孩子
    
    // 把 y 的左子树给 x 的右子树
    x.right = y.left;
    if (y.left !== null) {
        y.left.parent = x;
    }
    
    // 设置 y 的父节点
    y.parent = x.parent;
    
    if (x.parent === null) {
        // x 是根节点
        this.root = y;
    } else if (x === x.parent.left) {
        x.parent.left = y;
    } else {
        x.parent.right = y;
    }
    
    // x 变成 y 的左孩子
    y.left = x;
    x.parent = y;
}
```

#### 右旋（Right Rotate）

```
旋转前：
      y
     /
    x
     \
      b

以 y 为支点右旋：

    x
     \
      y
     /
    b

详细过程：
1. x 成为新的根
2. y 变成 x 的右孩子
3. x 原来的右孩子 b 变成 y 的左孩子

代码实现：
```

```javascript
/**
 * 右旋
 * @param {TreeNode} y - 旋转的支点节点
 */
function rightRotate(y) {
    const x = y.left;  // x 是 y 的左孩子
    
    // 把 x 的右子树给 y 的左子树
    y.left = x.right;
    if (x.right !== null) {
        x.right.parent = y;
    }
    
    // 设置 x 的父节点
    x.parent = y.parent;
    
    if (y.parent === null) {
        // y 是根节点
        this.root = x;
    } else if (y === y.parent.left) {
        y.parent.left = x;
    } else {
        y.parent.right = x;
    }
    
    // y 变成 x 的右孩子
    x.right = y;
    y.parent = x;
}
```

#### 旋转的特点

```
✓ 保持二叉搜索树的性质
  （左 < 根 < 右的顺序不变）

✓ 只改变局部结构
  （不影响其他部分）

✓ 时间复杂度：O(1)
  （只是改几个指针）

✓ 用于重新平衡
  （插入和删除时会用到）
```

### 13.4 插入操作 - 最复杂的部分

#### 插入的基本思想

```
第 1 步：像普通 BST 一样插入
  找到位置，插进去
  新节点标记为红色（为什么是红色？后面说）

第 2 步：检查是否违反规则
  主要是规则 4（红不相邻）
  如果违反了，就修复

第 3 步：通过变色和旋转来修复
  有 3 种情况（叔叔是红色、叔叔是黑色且侄子在右边、叔叔是黑色且侄子在左边）
```

#### 完整的插入代码

```javascript
/**
 * 红黑树节点
 */
class RBNode {
    constructor(val) {
        this.val = val;
        this.color = 'RED';  // 新节点默认红色
        this.left = null;
        this.right = null;
        this.parent = null;
    }
}

/**
 * 红黑树
 */
class RedBlackTree {
    constructor() {
        this.root = null;
        this.NIL = new RBNode(null);  // NIL 叶子
        this.NIL.color = 'BLACK';
    }
    
    /**
     * 插入节点
     */
    insert(val) {
        const newNode = new RBNode(val);
        newNode.left = this.NIL;
        newNode.right = this.NIL;
        
        // 第 1 步：像普通 BST 一样插入
        let parent = null;
        let current = this.root;
        
        while (current !== this.NIL && current !== null) {
            parent = current;
            if (val < current.val) {
                current = current.left;
            } else {
                current = current.right;
            }
        }
        
        newNode.parent = parent;
        
        if (parent === null) {
            // 空树，新节点是根
            this.root = newNode;
        } else if (val < parent.val) {
            parent.left = newNode;
        } else {
            parent.right = newNode;
        }
        
        // 第 2 步：修复可能违反的规则
        this.fixInsert(newNode);
    }
    
    /**
     * 修复插入后的红黑树性质
     */
    fixInsert(node) {
        // 情况 0：如果是根节点，染成黑色
        if (node.parent === null) {
            node.color = 'BLACK';
            return;
        }
        
        // 只要父节点是红色，就需要修复
        while (node.parent && node.parent.color === 'RED') {
            
            // 父节点是祖父的左孩子
            if (node.parent === node.parent.parent.left) {
                const uncle = node.parent.parent.right;  // 叔叔节点
                
                // 情况 1：叔叔是红色
                if (uncle && uncle.color === 'RED') {
                    // 父和叔都变黑
                    node.parent.color = 'BLACK';
                    uncle.color = 'BLACK';
                    // 祖父变红
                    node.parent.parent.color = 'RED';
                    // 继续检查祖父
                    node = node.parent.parent;
                } 
                // 情况 2：叔叔是黑色，且节点在右边
                else if (node === node.parent.right) {
                    // 左旋父节点
                    node = node.parent;
                    this.leftRotate(node);
                    // 转化成情况 3
                }
                // 情况 3：叔叔是黑色，且节点在左边
                else {
                    // 父变黑，祖父变红
                    node.parent.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    // 右旋祖父
                    this.rightRotate(node.parent.parent);
                }
            }
            // 父节点是祖父的右孩子（对称情况）
            else {
                const uncle = node.parent.parent.left;
                
                if (uncle && uncle.color === 'RED') {
                    node.parent.color = 'BLACK';
                    uncle.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    node = node.parent.parent;
                } else if (node === node.parent.left) {
                    node = node.parent;
                    this.rightRotate(node);
                } else {
                    node.parent.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    this.leftRotate(node.parent.parent);
                }
            }
        }
        
        // 最后确保根是黑色
        this.root.color = 'BLACK';
    }
    
    // 左旋和右旋方法（前面已给出）
    leftRotate(x) { /* ... */ }
    rightRotate(y) { /* ... */ }
}

// 测试
const rbt = new RedBlackTree();
rbt.insert(5);
rbt.insert(3);
rbt.insert(7);
rbt.insert(1);
rbt.insert(9);

console.log('红黑树构建成功！');
```

### 13.5 删除操作（简单了解）

```
删除比插入更复杂！

基本步骤：
1. 像普通 BST 一样删除
2. 如果删除的是黑色节点，会破坏平衡

有 8 种情况要处理！

```

### 13.6 为什么新节点是红色？

```
问题：为什么插入时默认是红色而不是黑色？

答案：为了少违反规则！

如果插入黑色：
→ 违反规则 5（黑高不同）
→ 影响整条路径
→ 很难修复

如果插入红色：
→ 最多违反规则 4（红相邻）
→ 只影响局部
→ 容易修复

所以选择红色！
```

### 13.7 红黑树的时间复杂度

```
定理：
红黑树的高度 ≤ 2 × log₂(n+1)

证明思路：
1. 设黑高为 bh
2. 根据规则 4，红色节点不超过黑色节点
3. 所以高度 h ≤ 2bh
4. 根据规则 5，至少有 2^bh - 1 个节点
5. 推出 h ≤ 2log₂(n+1)

推论：
查找：O(log n)
插入：O(log n)（包括旋转）
删除：O(log n)

非常稳定！
```

### 13.8 红黑树 vs AVL 树

| 特性 | 红黑树 | AVL 树 |
|------|--------|--------|
| 平衡性 | 大致平衡 | 严格平衡 |
| 查找性能 | O(log n) | O(log n) |
| 插入性能 | O(log n)，旋转少 | O(log n)，旋转多 |
| 删除性能 | O(log n) | O(log n) |
| 适用场景 | 插入删除频繁 | 查找为主 |
| 实现难度 | 较难 | 更难 |
| 实际应用 | C++ STL, Java TreeMap | 数据库索引 |

### 13.9 完整代码模板（精简版）

```javascript
class RBNode {
    constructor(val) {
        this.val = val;
        this.color = 'RED';
        this.left = null;
        this.right = null;
        this.parent = null;
    }
}

class RedBlackTree {
    constructor() {
        this.root = null;
        this.NIL = new RBNode(null);
        this.NIL.color = 'BLACK';
    }
    
    insert(val) {
        const node = new RBNode(val);
        node.left = this.NIL;
        node.right = this.NIL;
        
        // BST 插入
        let parent = null;
        let curr = this.root;
        while (curr !== this.NIL && curr !== null) {
            parent = curr;
            curr = val < curr.val ? curr.left : curr.right;
        }
        
        node.parent = parent;
        if (!parent) this.root = node;
        else if (val < parent.val) parent.left = node;
        else parent.right = node;
        
        this.fixInsert(node);
    }
    
    fixInsert(node) {
        while (node.parent && node.parent.color === 'RED') {
            if (node.parent === node.parent.parent.left) {
                const uncle = node.parent.parent.right;
                if (uncle && uncle.color === 'RED') {
                    node.parent.color = 'BLACK';
                    uncle.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    node = node.parent.parent;
                } else {
                    if (node === node.parent.right) {
                        node = node.parent;
                        this.leftRotate(node);
                    }
                    node.parent.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    this.rightRotate(node.parent.parent);
                }
            } else {
                // 对称情况
                const uncle = node.parent.parent.left;
                if (uncle && uncle.color === 'RED') {
                    node.parent.color = 'BLACK';
                    uncle.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    node = node.parent.parent;
                } else {
                    if (node === node.parent.left) {
                        node = node.parent;
                        this.rightRotate(node);
                    }
                    node.parent.color = 'BLACK';
                    node.parent.parent.color = 'RED';
                    this.leftRotate(node.parent.parent);
                }
            }
        }
        this.root.color = 'BLACK';
    }
    
    leftRotate(x) { /* 见前面代码 */ }
    rightRotate(y) { /* 见前面代码 */ }
}
```
