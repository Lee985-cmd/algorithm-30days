## 第 23 章：任意两点间的距离 - 所有点对最短路径

### 23.1 什么是最短路径问题？

#### 生活中的例子

**场景 1：地图导航**

```
你有一个全国地图

想知道任意两个城市之间的最短距离

北京到上海：1200km
北京到广州：2000km
上海到广州：1400km
...

这就是所有点对最短路径问题！
```

**场景 2：网络路由**

```
互联网有无数个路由器

每个路由器要知道：
到所有其他路由器的最短路径

这样数据包才能快速到达目的地！

这也是所有点对最短路径问题！
```

**说人话定义：**

```
所有点对最短路径 = 找图中任意两点间的最短距离

输入：
- 一个图 G=(V,E)
- 每条边有权重 w(u,v)

输出：
- 对所有顶点对 (u,v)，计算 d(u,v)
  从 u 到 v 的最短距离

解决方案：
✓ Floyd-Warshall 算法（动态规划）
✓ Johnson 算法（重新赋权 + Dijkstra）
```

### 23.2 Floyd-Warshall 算法 - 动态规划

#### 核心思想

```
生活比喻：不断尝试走中间点

Floyd-Warshall 的做法：
1. 初始时，dist[i][j] = 直接相连的边的权重
2. 依次考虑每个顶点 k 作为中间点
3. 对于每对顶点 (i,j)：
   问：从 i 到 j，走 i→k→j 是不是更近？
   如果是，更新 dist[i][j]
4. 重复 2-3，直到考虑了所有顶点

关键洞察：
如果 i 到 j 的最短路径经过 k
那么这条路径由两段组成：
- i 到 k 的最短路径
- k 到 j 的最短路径

这是最优子结构！可以用动态规划！
```

#### 代码实现

```javascript
/**
 * Floyd-Warshall 算法 - 所有点对最短路径
 * @param {number[][]} graph - 邻接矩阵表示的图
 * @param {number} INF - 无穷大值
 * @returns {number[][]} - 所有点对的最短距离
 */
function floydWarshall(graph, INF = Infinity) {
    const n = graph.length;
    
    // 初始化距离矩阵
    const dist = graph.map(row => [...row]);
    
    // 自己到自己的距离是 0
    for (let i = 0; i < n; i++) {
        dist[i][i] = 0;
    }
    
    // 动态规划：依次考虑每个中间点 k
    for (let k = 0; k < n; k++) {
        // 对于每对顶点 (i, j)
        for (let i = 0; i < n; i++) {
            for (let j = 0; j < n; j++) {
                // 如果走 i→k→j 更近，就更新
                if (dist[i][k] !== INF && dist[k][j] !== INF) {
                    const newDist = dist[i][k] + dist[k][j];
                    if (newDist < dist[i][j]) {
                        dist[i][j] = newDist;
                    }
                }
            }
        }
    }
    
    return dist;
}

// 测试
const INF = Infinity;
const graph = [
    [0,   3,   8, INF,  -4],
    [INF, 0, INF,   1,   7],
    [INF, 4,   0, INF, INF],
    [2, INF,  -5,   0, INF],
    [INF, INF, INF,   6,   0]
];

const dist = floydWarshall(graph, INF);
console.log('所有点对最短距离:');
console.table(dist);
```

### 23.3 Johnson 算法 - 稀疏图的优化

#### 核心思想

```
Johnson 算法的做法：
1. 添加一个虚拟源点 s，连向所有顶点，权重为 0
2. 用 Bellman-Ford 算法计算从 s 到所有点的最短距离 h[v]
3. 重新赋权：w'(u,v) = w(u,v) + h[u] - h[v]
   （这样所有权值都变成非负的）
4. 对每个顶点运行 Dijkstra 算法
5. 还原真实距离：d(u,v) = d'(u,v) - h[u] + h[v]

为什么要重新赋权？
→ Dijkstra 不能处理负权边
→ 重新赋权后全是正权
→ 可以用更快的 Dijkstra！

巧妙之处：
重新赋权不改变最短路径！
只是给所有路径加了一个常数
```

#### 代码实现

```javascript
/**
 * Johnson 算法 - 所有点对最短路径
 */
function johnson(graph) {
    const n = graph.length;
    const INF = Infinity;
    
    // 第 1 步：Bellman-Ford 计算势函数 h
    const h = bellmanFordForJohnson(graph);
    
    if (h === null) {
        console.log('图中存在负权环！');
        return null;
    }
    
    // 第 2 步：重新赋权
    const reweighted = Array(n).fill(null).map(() => Array(n).fill(INF));
    for (let u = 0; u < n; u++) {
        for (let v = 0; v < n; v++) {
            if (graph[u][v] !== INF) {
                reweighted[u][v] = graph[u][v] + h[u] - h[v];
            }
        }
    }
    
    // 第 3 步：对每个顶点运行 Dijkstra
    const dist = Array(n).fill(null).map(() => Array(n).fill(INF));
    
    for (let s = 0; s < n; s++) {
        const d = dijkstra(reweighted, s);
        
        // 还原真实距离
        for (let t = 0; t < n; t++) {
            if (d[t] !== INF) {
                dist[s][t] = d[t] - h[s] + h[t];
            }
        }
    }
    
    return dist;
}

/**
 * Bellman-Ford 用于 Johnson 算法
 */
function bellmanFordForJohnson(graph) {
    const n = graph.length;
    const INF = Infinity;
    const h = new Array(n).fill(0);  // 初始为 0
    
    // 松弛 n 次
    for (let i = 0; i < n; i++) {
        for (let u = 0; u < n; u++) {
            for (let v = 0; v < n; v++) {
                if (graph[u][v] !== INF) {
                    if (h[u] + graph[u][v] < h[v]) {
                        h[v] = h[u] + graph[u][v];
                    }
                }
            }
        }
    }
    
    // 检查负权环
    for (let u = 0; u < n; u++) {
        for (let v = 0; v < n; v++) {
            if (graph[u][v] !== INF) {
                if (h[u] + graph[u][v] < h[v]) {
                    return null;  // 有负权环
                }
            }
        }
    }
    
    return h;
}

/**
 * Dijkstra 算法
 */
function dijkstra(graph, start) {
    const n = graph.length;
    const INF = Infinity;
    const dist = new Array(n).fill(INF);
    const visited = new Array(n).fill(false);
    
    dist[start] = 0;
    
    for (let count = 0; count < n; count++) {
        let u = -1, minDist = INF;
        for (let v = 0; v < n; v++) {
            if (!visited[v] && dist[v] < minDist) {
                minDist = dist[v];
                u = v;
            }
        }
        
        if (u === -1) break;
        visited[u] = true;
        
        for (let v = 0; v < n; v++) {
            if (graph[u][v] !== INF && !visited[v]) {
                const newDist = dist[u] + graph[u][v];
                if (newDist < dist[v]) {
                    dist[v] = newDist;
                }
            }
        }
    }
    
    return dist;
}

// 测试
const graph = [
    [0,   3,   8, INF,  -4],
    [INF, 0, INF,   1,   7],
    [INF, 4,   0, INF, INF],
    [2, INF,  -5,   0, INF],
    [INF, INF, INF,   6,   0]
];

const dist = johnson(graph);
console.log('Johnson 算法结果:');
console.table(dist);
```

#### 时间复杂度

```
Bellman-Ford: O(VE)
n 次 Dijkstra: O(n(E log n))（用优先队列）

总时间：O(VE + nE log n)

对于稀疏图（E << n²）：
≈ O(nE log n)

比 Floyd 的 O(n³) 快！

对于稠密图（E ≈ n²）：
≈ O(n³ log n)

比 Floyd 慢...

所以：
✓ 稀疏图用 Johnson
✓ 稠密图用 Floyd
```

### 23.6 完整代码模板（背诵版）

```javascript
/**
 * Floyd-Warshall 算法
 */
function floydWarshall(graph, INF = Infinity) {
    const n = graph.length;
    const dist = graph.map(row => [...row]);
    
    for (let i = 0; i < n; i++) {
        dist[i][i] = 0;
    }
    
    for (let k = 0; k < n; k++) {
        for (let i = 0; i < n; i++) {
            for (let j = 0; j < n; j++) {
                if (dist[i][k] !== INF && dist[k][j] !== INF) {
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
    }
    
    return dist;
}

/**
 * Johnson 算法（简化版）
 */
function johnson(graph) {
    const n = graph.length;
    const INF = Infinity;
    
    // Bellman-Ford 计算势函数
    const h = new Array(n).fill(0);
    for (let i = 0; i < n; i++) {
        for (let u = 0; u < n; u++) {
            for (let v = 0; v < n; v++) {
                if (graph[u][v] !== INF && h[u] + graph[u][v] < h[v]) {
                    h[v] = h[u] + graph[u][v];
                }
            }
        }
    }
    
    // 重新赋权 + Dijkstra
    const dist = Array(n).fill(null).map(() => Array(n).fill(INF));
    
    for (let s = 0; s < n; s++) {
        const d = new Array(n).fill(INF);
        d[s] = 0;
        const visited = new Array(n).fill(false);
        
        for (let count = 0; count < n; count++) {
            let u = -1, minVal = INF;
            for (let v = 0; v < n; v++) {
                if (!visited[v] && d[v] < minVal) {
                    minVal = d[v];
                    u = v;
                }
            }
            
            if (u === -1) break;
            visited[u] = true;
            
            for (let v = 0; v < n; v++) {
                if (graph[u][v] !== INF && !visited[v]) {
                    const w = graph[u][v] + h[u] - h[v];
                    if (d[u] + w < d[v]) {
                        d[v] = d[u] + w;
                    }
                }
            }
        }
        
        for (let t = 0; t < n; t++) {
            if (d[t] !== INF) {
                dist[s][t] = d[t] - h[s] + h[t];
            }
        }
    }
    
    return dist;
}
```
