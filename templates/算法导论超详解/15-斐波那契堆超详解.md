### 18.1 什么是斐波那契堆？为什么这么厉害？

#### 生活中的例子

**场景 1：多个任务队列合并**

```
你有三个待办事项清单：
- 工作清单：10 个任务
- 学习清单：8 个任务
- 生活清单：5 个任务

现在想把它们合并成一个大清单
并且快速找到优先级最高的任务

普通方法：
把所有任务倒在一起，重新排序 → O(n)

斐波那契堆：
直接把三个链表连起来 → O(1)
找最高优先级 → O(1)

这就是斐波那契堆的优势！
```

**场景 2：图算法中的优先队列**

```
Dijkstra 算法需要一个优先队列

操作包括：
- 插入新节点：O(log n)
- 取出最小值：O(log n)
- 减小某个节点的键值：O(log n)

如果用斐波那契堆：
- 插入：O(1) ✓
- 取出最小值：O(log n)
- 减小键值：O(1) ✓

整体能加速 Dijkstra 算法！
```

```
斐波那契堆 = 一种可合并的最小堆

特点：
```

### 18.2 斐波那契堆的结构

#### 核心思想

```
斐波那契堆由多棵树组成（森林）

每棵树都是"最小堆性质"：
- 父节点的键值 ≤ 子节点的键值

树之间通过循环双向链表连接

有一个指针指向最小节点

关键特性：
✓ 树的度数（孩子数量）不会太大
✓ 延迟合并（懒惰策略）
✓ 摊还分析保证效率
```

#### 节点结构

```javascript
/**
 * 斐波那契堆节点
 */
class FibNode {
    constructor(key) {
        this.key = key;           // 键值
        this.degree = 0;          // 孩子的数量
        this.parent = null;       // 父节点
        this.child = null;        // 第一个孩子
        this.left = null;         // 左兄弟（循环链表）
        this.right = null;        // 右兄弟（循环链表）
        this.mark = false;        // 标记位（是否失去过孩子）
    }
}

/**
 * 斐波那契堆
 */
class FibonacciHeap {
    constructor() {
        this.min = null;  // 指向最小节点
        this.n = 0;       // 节点总数
    }
    
    /**
     * 插入节点 - O(1)
     */
    insert(key) {
        const node = new FibNode(key);
        
        // 把新节点加入根表
        if (this.min === null) {
            this.min = node;
            node.left = node;
            node.right = node;
        } else {
            // 插入到 min 前面
            node.right = this.min;
            node.left = this.min.left;
            this.min.left.right = node;
            this.min.left = node;
            
            // 更新 min
            if (node.key < this.min.key) {
                this.min = node;
            }
        }
        
        this.n++;
        return node;
    }
    
    /**
     * 查找最小值 - O(1)
     */
    minimum() {
        return this.min;
    }
}

// 测试
const heap = new FibonacciHeap();
heap.insert(10);
heap.insert(5);
heap.insert(15);

console.log('最小值:', heap.minimum().key);  // 5 ✓
```

#### 详细结构图

```
一个斐波那契堆：

        1(min)
       / | \
      3  5  7
     / \
    8   9

根表（循环链表）：
1 ↔ 4 ↔ 6 ↔ 10

其中：
- 1 有 3 个孩子：3, 5, 7
- 3 有 2 个孩子：8, 9
- 4, 6, 10 是单节点

min 指针指向 1 ✓
```

### 18.3 合并操作 - O(1) 的核心

#### 核心思想

```
斐波那契堆最厉害的地方：合并两个堆只要 O(1)！

做法超级简单：
把两个根表连起来
选较小的作为新的 min

就这么简单！
```

#### 代码实现

```javascript
class FibonacciHeap {
    // ... 前面的代码 ...
    
    /**
     * 合并两个堆 - O(1)
     */
    union(otherHeap) {
        if (otherHeap.min === null) return;  // 对方是空堆
        
        if (this.min === null) {
            // 自己是空堆
            this.min = otherHeap.min;
        } else {
            // 连接两个根表
            const temp = this.min.right;
            
            this.min.right = otherHeap.min;
            otherHeap.min.left.right = temp;
            temp.left = otherHeap.min.left;
            otherHeap.min.left = this.min;
            
            // 更新 min
            if (otherHeap.min.key < this.min.key) {
                this.min = otherHeap.min;
            }
        }
        
        this.n += otherHeap.n;
    }
}

// 测试
const heap1 = new FibonacciHeap();
heap1.insert(5);
heap1.insert(10);

const heap2 = new FibonacciHeap();
heap2.insert(3);
heap2.insert(8);

heap1.union(heap2);

console.log('合并后的最小值:', heap1.minimum().key);  // 3 ✓
```

#### 合并过程图解

```
合并前：

heap1:  5(min) ↔ 10
heap2:  3(min) ↔ 8

═══════════════════════════════════
合并后
═══════════════════════════════════

把两个环连起来：
5 ↔ 10 ↔ 3 ↔ 8
↑
min（因为 3 < 5，所以指向 3）

重新整理：
3(min) ↔ 8 ↔ 5 ↔ 10

完成！O(1) 时间！✓
```

### 18.4 提取最小值 - 最复杂的操作

#### 核心思想

```
extract-min 是最复杂的操作！

步骤：
1. 取出 min 节点
2. 把 min 的所有孩子加入根表
3. 合并度数相同的树（consolidate）
4. 更新 min

第 3 步是关键：
用数组记录每个度数的树
如果有两棵树度数相同，就合并它们
保持每个度数最多一棵树
```

#### 代码实现

```javascript
class FibonacciHeap {
    // ... 前面的代码 ...
    
    /**
     * 提取最小值 - O(log n) 摊还
     */
    extractMin() {
        const z = this.min;
        
        if (z === null) return null;  // 空堆
        
        // 把 min 的所有孩子加入根表
        if (z.child !== null) {
            let child = z.child;
            do {
                const next = child.right;
                
                // 从孩子的链表中移除
                child.parent = null;
                
                // 加入根表
                child.right = z;
                child.left = z.left;
                z.left.right = child;
                z.left = child;
                
                child = next;
            } while (child !== z.child);
        }
        
        // 从根表移除 z
        z.left.right = z.right;
        z.right.left = z.left;
        
        // 更新 min
        if (z === z.right) {
            this.min = null;  // 堆空了
        } else {
            this.min = z.right;
            // 合并度数相同的树
            this.consolidate();
        }
        
        this.n--;
        return z;
    }
    
    /**
     * 合并度数相同的树
     */
    consolidate() {
        const maxDegree = Math.floor(Math.log2(this.n)) + 1;
        const roots = new Array(maxDegree).fill(null);
        
        // 遍历根表
        let nodesToProcess = [];
        let current = this.min;
        do {
            nodesToProcess.push(current);
            current = current.right;
        } while (current !== this.min);
        
        for (let w of nodesToProcess) {
            let x = w;
            let d = x.degree;
            
            while (roots[d] !== null) {
                let y = roots[d];
                
                // 确保 x 是较小的那个
                if (x.key > y.key) {
                    [x, y] = [y, x];
                }
                
                // 把 y 链接到 x 下
                this.link(y, x);
                roots[d] = null;
                d++;
            }
            
            roots[d] = x;
        }
        
        // 重建根表
        this.min = null;
        for (let root of roots) {
            if (root !== null) {
                this.insertRoot(root);
            }
        }
    }
    
    /**
     * 链接操作：把 y 挂到 x 下
     */
    link(y, x) {
        // 从根表移除 y
        y.left.right = y.right;
        y.right.left = y.left;
        
        // 设置 y 为 x 的孩子
        y.parent = x;
        
        if (x.child === null) {
            x.child = y;
            y.left = y;
            y.right = y;
        } else {
            y.right = x.child;
            y.left = x.child.left;
            x.child.left.right = y;
            x.child.left = y;
        }
        
        x.degree++;
        y.mark = false;
    }
    
    /**
     * 插入节点到根表（不更新 min）
     */
    insertRoot(node) {
        if (this.min === null) {
            this.min = node;
            node.left = node;
            node.right = node;
        } else {
            node.right = this.min;
            node.left = this.min.left;
            this.min.left.right = node;
            this.min.left = node;
            
            if (node.key < this.min.key) {
                this.min = node;
            }
        }
    }
}

// 测试
const heap = new FibonacciHeap();
heap.insert(5);
heap.insert(3);
heap.insert(8);
heap.insert(1);

console.log('提取最小值:', heap.extractMin().key);  // 1 ✓
console.log('现在的最小值:', heap.minimum().key);   // 3 ✓
```

### 18.5 减小键值 - O(1) 的魔法

#### 核心思想

```
decrease-key 操作：
把某个节点的键值减小

如果破坏了堆性质（比父节点小了）：
1. 把这个节点切下来，放到根表
2. 检查父节点：
   - 如果父节点没失去过孩子，标记它
   - 如果父节点已经失去过孩子，把它也切下来
   - 继续向上检查

这叫"级联剪切"！
```

#### 代码实现

```javascript
class FibonacciHeap {
    // ... 前面的代码 ...
    
    /**
     * 减小键值 - O(1) 摊还
     */
    decreaseKey(node, newKey) {
        if (newKey > node.key) {
            throw new Error('新键值必须更小');
        }
        
        node.key = newKey;
        const parent = node.parent;
        
        // 如果违反堆性质，或者没有父节点
        if (parent !== null && node.key < parent.key) {
            this.cut(node, parent);
            this.cascadingCut(parent);
        }
        
        // 更新 min
        if (node.key < this.min.key) {
            this.min = node;
        }
    }
    
    /**
     * 剪切操作
     */
    cut(x, y) {
        // 从 y 的孩子中移除 x
        if (y.degree === 1) {
            y.child = null;
        } else {
            x.left.right = x.right;
            x.right.left = x.left;
            
            if (y.child === x) {
                y.child = x.right;
            }
        }
        
        y.degree--;
        
        // 把 x 加入根表
        x.parent = null;
        x.left.right = this.min;
        x.right = this.min.right;
        this.min.right.left = x;
        this.min.right = x;
        
        x.mark = false;
    }
    
    /**
     * 级联剪切
     */
    cascadingCut(y) {
        const parent = y.parent;
        
        if (parent !== null) {
            if (!y.mark) {
                // 第一次失去孩子，标记
                y.mark = true;
            } else {
                // 已经失去过孩子，剪切
                this.cut(y, parent);
                this.cascadingCut(parent);
            }
        }
    }
    
    /**
     * 删除节点（通过减小到 -∞ 再提取）
     */
    delete(node) {
        this.decreaseKey(node, -Infinity);
        this.extractMin();
    }
}

// 测试
const heap = new FibonacciHeap();
const node5 = heap.insert(5);
heap.insert(3);
heap.insert(8);

console.log('减小前最小值:', heap.minimum().key);  // 3 ✓

heap.decreaseKey(node5, 1);

console.log('减小后最小值:', heap.minimum().key);  // 1 ✓
```

### 18.7 实际应用

#### 应用 1：Dijkstra 算法优化

```javascript
/**
 * 用斐波那契堆优化的 Dijkstra
 */
function dijkstraFib(graph, start) {
    const n = graph.length;
    const dist = new Array(n).fill(Infinity);
    const prev = new Array(n).fill(null);
    
    const heap = new FibonacciHeap();
    const nodes = new Map();  // 顶点到堆节点的映射
    
    // 初始化
    dist[start] = 0;
    nodes.set(start, heap.insert(0));
    
    for (let i = 0; i < n; i++) {
        if (i !== start) {
            nodes.set(i, heap.insert(Infinity));
        }
    }
    
    while (heap.minimum().key !== Infinity) {
        const minNode = heap.extractMin();
        const u = getKeyByValue(nodes, minNode);
        
        for (let v = 0; v < n; v++) {
            if (graph[u][v] > 0) {
                const alt = dist[u] + graph[u][v];
                if (alt < dist[v]) {
                    dist[v] = alt;
                    prev[v] = u;
                    heap.decreaseKey(nodes.get(v), alt);
                }
            }
        }
    }
    
    return { dist, prev };
}

function getKeyByValue(map, value) {
    for (let [key, val] of map.entries()) {
        if (val === value) return key;
    }
    return null;
}

// 时间复杂度：O(V + E log V)
// 比普通 Dijkstra 的 O((V+E) log V) 更快！
```
